# Textract Privacy Policy

Effective date: April 25, 2026

Textract is a browser extension that extracts text from images, video frames, and user-selected screen regions using local OCR.

## Data Collection

Textract does **not** collect, store, or transmit personal data or website content to any external server.

## How Processing Works

- OCR processing is performed locally in the extension runtime on the user's device.
- Images, video frames, and selected regions are used only to generate text for the user.
- Extracted text is shown in the extension panel for user review and manual editing.

## Clipboard Access

Textract writes to the clipboard only after explicit user action, such as clicking "Copy All".

## Third-Party Sharing

Textract does not sell, share, or transfer user data to third parties.

## Tracking and Analytics

Textract does not use advertising SDKs, analytics trackers, or remote code to collect user data.

## Policy Updates

If Textract's data handling changes in the future, this policy will be updated before those changes are released.
